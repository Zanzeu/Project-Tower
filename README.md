运行 Tower.exe 

点击Start开始游戏

# Unity版本

2022.3.57f1c2

# 键位

鼠标操作

# 已实现清单

NavMesh实现寻路，同时可以进行路径拐弯

可以通过Json进行关卡配置，包括生成间隔，敌人，路径点，波次

可以通过Json进行敌人、塔防、Buff的配置

实现了3种塔：

    1.弓箭塔：攻速快，单体，射程远

    2.炸弹塔：炸弹AoE，攻速慢，射程近

    3.减益塔：施加范围减速、易伤、燃烧

    升级：

    增加攻击速度、伤害、射程，增加额外效果

实现了6种敌人：

    第一类：死亡召唤低级

    第二类：死亡为所有敌人施加加速buff

    第三类：死亡冻结周围塔防，不会进行行动

    第四类：boss，免疫燃烧和易伤，死亡召唤

Buff实现：

    减速：减缓敌人移动速度

    燃烧：周期对敌人造成伤害Dot，可以叠层，基于层数造成额外伤害

    易伤：增加受到伤害，可以叠层，基于层数增加额外伤害

    冻结：塔防一定时间内无法攻击

经济系统：

    建造和升级需要消耗金币

    击杀敌人获得金币

存档：

    记录未遇见过的敌人，初次见到显示图鉴，之后永久加入图鉴中

UI：

    显示塔防信息和敌人信息，倍速控制，暂停

其他实现：

数据都可以用Json进行配置，通过用Odin实现了自定义编辑器，可以对Json进行可视化编辑

用坐标距离进行碰撞检测，可以适应高压环境
